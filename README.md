# Overview

This project is an interactive Power BI Dashboard developed to analyze and visualize data effectively. It transforms raw data into meaningful insights using data modeling, DAX calculations, and dynamic visualizations.

The dashboard helps users explore trends, compare performance metrics, and make data-driven decisions through interactive reports.
